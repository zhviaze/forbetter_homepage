# ForBetter Homepage

ForBetter合同会社のコーポレートサイトです。ビルド不要の静的サイトとして管理しています。

## ファイル構成

- `index.html` - ページ本体
- `styles.css` - レイアウトとデザイン
- `script.js` - モバイルメニューと年表示
- `assets/forbetter-logo.png` - ForBetterロゴ
- `assets/forbetter-logo-trimmed.png` - 表示用に余白を調整したForBetterロゴ
- `assets/hero-fintech-consulting.png` - ヒーロー写真

## ローカル確認

ブラウザで `index.html` を開くか、任意の静的サーバーで配信してください。

```bash
python3 -m http.server 8080
```

上記コマンドを `forbetter_homepage` フォルダ内で実行した場合、`http://localhost:8080` で確認できます。
